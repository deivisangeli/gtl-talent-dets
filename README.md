# GTL — Talent Determinants

This repository contains the code, data, and outputs for the **Wikipedia Database project** — an analysis of the impact of scientific infrastructure on talent and innovation outcomes, using historical data.

## Repository Structure

```
Jan26/
├── Country/        # Country-level analysis (initial approach)
├── HYDE/           # HYDE population data analysis
│   ├── Code US/    # US-level HYDE analysis
│   ├── County/     # US county-level analysis
│   ├── Input/      # Raw HYDE data
│   ├── Output/     # Processed data
│   └── Results*/   # Figures and tables
└── Project/        # Main project
    ├── Code/       # R scripts
    ├── Input/      # Raw data
    ├── Output/     # Processed data
    └── Results/    # Figures and tables
```

## Notes

- The **Country analysis** was the first approach to estimating the impact of scientific infrastructure construction.
- The **HYDE analysis** uses HYDE historical population data for the estimation, at the world, US state, and US county levels.
- `Jan26/scientific_facilities (1).xlsx` and `Jan26/Wikipedia Project.pdf` contain the final facility list and project report.
